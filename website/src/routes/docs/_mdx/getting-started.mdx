按照以下步骤快速开始使用组件库。

## 先决条件

在开始之前，请确保您已经在项目中安装了Tailwind CSS。请先阅读 [Tailwind CSS 安装指南](https://tailwindcss.com/docs/installation/using-vite)

## 安装

通过以下任一方式安装 Resolid React UI 组件包：

<CodeGroup>

```bash [pnpm]
pnpm add @resolid/react-ui
```

```bash [npm]
npm install @resolid/react-ui
```

```bash [yarn]
yarn add @resolid/react-ui
```

```bash [bun]
bun add @resolid/react-ui
```

</CodeGroup>

## 设置 CSS

在你的 CSS 中导入 Tailwind CSS 和 Resolid React UI

```css
@import "tailwindcss";
@import "@resolid/react-ui/tailwindcss";
```

## 设置 Provider

在您的应用程序的 `root` 添加 `ConfigProvider` 至关重要。

```jsx
// 1. 导入 `ConfigProvider`
import { ConfigProvider } from "@resolid/react-ui";

function Root() {
  // 2. 使用 ConfigProvider 包裹你的 App
  return (
    <ConfigProvider>
      <YourApp />
    </ConfigProvider>
  );
}
```

## 使用组件

导入您需要的组件并开始构建应用程序。

```jsx demo
import { Button } from "@resolid/react-ui";

export default function App() {
  return (
    <div className={"flex justify-center"}>
      <Button>你好</Button>
    </div>
  );
}
```
